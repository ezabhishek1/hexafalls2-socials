# HexaFalls 2 Socials Portal

The official gateway for **HexaFalls 2** (2026), JIS University's premier TechFest.

## Features
- **Wizarding World Aesthetics**: Customized UI featuring a minimal and elegant theme with a Diagon Alley backdrop.
- **HexaFalls Color System**: Implemented using our custom slate, navy, neon cyan, vivid orange, and warm cream palette.
- **Interactive Social Cards**: Hover-scale cards with custom handles, gradient icons, and arrow indicators.
- **Clean Performance**: Next.js static export deployed directly to GitHub Pages.
